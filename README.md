# My config for neovim

My simple configuration for neovim.

## Install

Copy the contents of `init.lua` into `~/.config/nvim/init.lua`.

This can also be done with the following command:
```bash
git clone https://github.com/RemyIsCool/neovim-config.git ~/.config/nvim
```
